# ShowHost
An Object Show Focused Animation Program.



# CONTROLS

* Scroll whell while selecting to resize objects perfectly
* Scroll whell + shift to scroll throught keyframes
* Ctrl + Z to undo
* Space to hit play
* Everything else is mouse controlled.

# How to import custom assets

1. Go to the "Open Assets Folder" option in the "File" Category (If the button does nothing, click "Export MP4 Video" first. that'll create the Asset Folder.)
2. Put your PNGs and WAVs on there! (ONLY THOSE FILE TYPES ARE ALLOWED)
3. Click "Import Asset (Type Name)" and type the file's name WITH THE EXTENSION
4. There should be an "imported" asset category or an audio playing now!
